# Exceptions

Uma excessão é quando o usuário atinge um caso fora do nosso "use case" do programa.

let soma = (a , b) -> a + b;

Func<T1, T2, R> soma = (a, b) => a + b;
Func<T, R> dobro = (x) => 2*x;

## assinatura 

double soma(double a, double b)

## implementação 

double soma(double a, double b) {
  return a + b;
}
